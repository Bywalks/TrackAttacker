# TrackAttacker V1.0 #

**TrackAttacker** 是一个批量通过IP追踪攻击者指纹工具

TrackAttacker目前版本为1.0，已满足功能如下

* 1：IP查域名
* 2：IP查地址
* 3：IP查端口
* 4：IP查主机名
* 5：域名查备案
* 6：域名查Whois

------

### Install ###

python3 -m pip install -r requirements.txt

### 开始使用 ###

```
python3 TrackAttack.py3
python3 TrackAttack.py3 all
```
